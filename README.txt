Druqs
http://drupal.org/project/druqs

DESCRIPTION
-----------
Druqs provides a quick search field for users, nodes, taxonomy terms and menu
items.

INSTALLATION
-----------
1. Copy module into your modules directory.
2. Enable the module at /admin/modules.
3. Manage configuration profiles at /admin/config/user-interface/druqs
4. If you're not using the admin toolbar, add the Druqs block at
/admin/structure/block
